# Hot Keys

| Key comb | In mode |  Effect |
|-------   | ------ | ------ |
| Ctrl - t | Normal |oggle NERDTree|
| [space]ev | Normal | edit vimrc|
| [space]sv | Normal | source vimrc|
| jj | Insert | Quit insert mode and save|
| [space]" | Normal | Surround the word with ""|
| [space]( | Normal | Surround the word with ()|
